
# 🛒 E-Commerce Web App

## 🚀 Live Demo
https://my-ecommerce-app-three-pink.vercel.app/

## 📌 Overview
A modern and responsive e-commerce web application that allows users to browse products, search by category, and manage their shopping cart and wishlist.

## 🛠 Tech Stack
- React.js
- Vite
- Tailwind CSS
- Zustand (State Management)
- REST API

## ✨ Features
- Product listing with categories and filters
- Search functionality
- Shopping cart with quantity management
- Wishlist system
- Persistent state (cart & wishlist)
- Fully responsive design
